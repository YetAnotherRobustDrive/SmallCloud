# How to run Server and Web UI
```bash
# 1. Clone this repository
git clone https://github.com/YetAnotherRobustDrive/SmallCloud

# 2. Move to the directory
cd SmallCloud

# 3. Create '.env' file and fill the env list
touch .env 

# 4. Run docker-compose
docker-compose up -d
```
